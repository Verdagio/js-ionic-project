To do app
How to use.

The app is made up of 2 tabs.

TODO TAB
In this tab you can veiw the tasks and delete them.
To delete a task swipe it to the left and then click
the delete button.

CREATE TAB
In this tab you can type the title and details of the
task and then click the Add task button below to add it
to the list of tasks that are displayed on the to-do tab.

All tasks will display the title, details and date the task
was added.
